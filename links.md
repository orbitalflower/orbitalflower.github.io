## Commodore Amiga

The greatest home computer ever made. Some of this information is also relevant to other 68000-based platforms, such as the Sega Mega Drive.

### AMOS Basic

An Amiga programming language which made it easier to produce games. It has since been released open source.

- [Amiga Tools by RETREAM](https://retream.itch.io/amiga-tools)<br>Various tools including an AMOS Pro executable optimizer.
- [AMOS Classic Documentation](https://gitlab.com/amigasourcecodepreservation/amos-classic-documentation)<br>The official manuals of the AMOS BASIC programming language family, including EASY AMOS, AMOS Pro, TOME, etc. (PDF)
- [AMOS Professional Manual](https://amospromanual.dev/)<br>AMOS Professional manual in HTML format.
- [amostools](https://github.com/kyz/amostools)<br>Tools to extract source code from .AMOS files.
- [F1 Licenseware Disk 55c](https://discmaster.textfiles.com/browse/562/F1%20Licenseware%20-%20Volume%201.iso/disks/055c.dms/055c.adf)<br>Various source codes for AMOS games.
- [Game Makers Manual AMOS](https://gitlab.com/amigasourcecodepreservation/game-makers-manual-amos)<br>Book on game development in various genres.
- [Writing RPGs in AMOS](https://gitlab.com/amigasourcecodepreservation/writing-rpgs-in-amos)<br>Book on writing RPGs.

### Blitz BASIC

A powerful Amiga programming language suited to both games and desktop applications. Development continues with a much improved open-source version called AmiBlitz.

- [2ndmouse.txt](https://discmaster.textfiles.com/view/1074/amigaformatcd18.iso/-seriously_amiga-/shareware/programming/blitzstuff/blitzfaq/faqlists/2ndmouse.txt)<br>A routine to use two mouse pointers at once for multiplayer mouse games.
- [AmiBlitz3](https://github.com/AmiBlitz/AmiBlitz3)<br>{% include hot.html %} The full Amiga programming language, including a modern IDE and numerous add-on modules.
- [AmiBlitz¬≥ Guide](https://web.archive.org/web/20210301220214/http://hd-rec.de/AmiBlitz3/main.html)<br>HTML version of AmiBlitz 3 manual, though oudated. (Archive)
- [Blitz Basic 2 Ultimate CD](https://github.com/binaryfoundry/BLITZCD)<br>The definitive Blitz Basic CD-ROM for Amiga.
- [Blitz Basic Tutorials](https://www.youtube.com/playlist?list=PLoEVvNYw8aQWvI2JVP6YEgNuXXwMwE_Gh)<br>YouTube playlist of Blitz Basic tutorials.
- [blitz2/tools/readbb2/scan](https://github.com/nitrologic/blitz2/tree/master/tools/readbb2/scan)<br>A list of Blitz Basic source code in plain text format.
